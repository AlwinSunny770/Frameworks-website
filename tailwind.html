Tailwind

Tailwind CSS is a free, open-source, and utility-first CSS framework that helps developers build custom user interfaces. It's a low-level framework that offers utility classes so developers can create their own reusable components. Tailwind CSS doesn't offer fully styled components like buttons, dropdowns, and navbars. Tailwind CSS is highly customizable and has over 500 components that can be used in projects. It's fast, flexible, consistent, scalable, and creative. Some companies that use Tailwind CSS include: OptimHire, Albertsons, Urban Insight, Intellibus. Tailwind CSS was originally written by Adam Wathan, Jonathan Reinink, David Hemphill, and Steve Schoger.
